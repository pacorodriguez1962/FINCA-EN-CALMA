## Cómo publicar en GitHub Pages

1. Crea una cuenta en [github.com](https://github.com)
2. Crea un repositorio nuevo llamado `fincaencalma`
3. Sube todos los archivos (index.html + fotos)
4. Ve a **Settings → Pages**
5. En *Source* selecciona **main** y carpeta **/ (root)**
6. Guarda — en 1 minuto tendrás tu URL
